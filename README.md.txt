# WPF Prodej Aut

Tento projekt je WPF aplikace pro zpracování a analýzu prodejů aut z XML souboru.

## 📌 Funkcionalita
✅ Umožňuje vybrat XML soubor s daty o prodaných autech.  
✅ Zobrazí seznam prodaných aut.  
✅ Sečte celkovou cenu vozů prodaných o víkendu.  
✅ Výsledky zobrazuje ve správném dvouřádkovém formátu.  

## 📂 Struktura projektu
- **MainWindow.xaml** → Uživatelské rozhraní
- **MainWindow.xaml.cs** → Logika aplikace
- **prodana_auta.xml** → Vzorový soubor s daty
- **README.md** → Tento popis projektu

## 🛠 Technologie
- **C#**
- **WPF**
- **XML**

## 🚀 Jak spustit aplikaci?
1. Stáhni repozitář a otevři projekt ve **Visual Studio**.
2. Spusť aplikaci pomocí **F5**.
3. Vyber soubor **`prodana_auta.xml`**.
4. Tabulky se naplní daty.
